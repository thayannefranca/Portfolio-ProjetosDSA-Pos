## Projetos

### Sistema de Recomendação de Candidatos - ATS

Desenvolvimento de sistema de recomendação para sugerir candidatos compatíveis com vagas abertas na plataforma ATS.

Principais características:

- Representação semântica de currículos e descrições de vagas com embeddings de texto;
- Utilização de BERT para análise textual;
- Cálculo de similaridade cosseno para identificar candidatos mais aderentes;
- Desenvolvimento de pipeline em Python para processamento periódico de novas vagas;
- Geração automática de recomendações para recrutadores.

O sistema permite identificar candidatos relevantes mesmo sem histórico prévio de candidatura, ampliando o pool de talentos disponível para recrutamento.

---

### Sistema de Triagem Automatizada de Candidatos - ATS

Desenvolvimento de sistema de triagem automática de candidatos utilizando Large Language Models para avaliar aderência aos critérios de uma vaga.

Principais características:

- Avaliação baseada em critérios definidos pelos recrutadores;
- Análise semântica de currículos com LLMs;
- Geração de justificativas para apoiar a avaliação;
- Desenvolvimento em Python;
- Deploy na plataforma Carol da TOTVS.

A solução contribui para padronizar avaliações iniciais e fornecer feedback mais claro aos candidatos.

---

### Sistema de Detecção de Anomalias em Folha de Pagamento

Desenvolvimento de sistema de auditoria automatizada para identificar inconsistências e valores atípicos em dados de folha de pagamento.

Principais características:

- Aplicação de algoritmos de detecção de anomalias, como Isolation Forest;
- Análise de padrões incomuns em variáveis financeiras;
- Desenvolvimento de pipeline de análise em Python;
- Utilização da biblioteca SHAP para interpretação dos modelos;
- Identificação das features responsáveis pelas anomalias detectadas.

A solução auxilia equipes de RH e auditoria na identificação automática de possíveis erros ou inconsistências nos pagamentos.

---

### Consultor Digital de RH

Desenvolvimento de assistente inteligente para análise de dados de RH utilizando arquitetura baseada em agentes e LLMs.

Principais características:

- Arquitetura multiagente com LangGraph;
- Agentes especializados para consulta, análise e geração de respostas;
- Integração com banco de dados corporativo via Trino;
- Execução automática de consultas SQL;
- Interpretação de perguntas em linguagem natural;
- Respostas voltadas para gestores e equipes de RH.

O sistema permite que gestores realizem consultas sobre colaboradores, métricas e indicadores de RH utilizando linguagem natural.

---

### Análise de Sentimento em Pesquisas de Engajamento

Desenvolvimento de pipeline de análise automática de comentários em pesquisas de clima organizacional.

Principais características:

- Uso de LLMs para extração automática de tópicos;
- Classificação de sentimentos em positivo, neutro e negativo;
- Identificação dos temas mais mencionados pelos colaboradores;
- Processamento de grandes volumes de respostas abertas;
- Geração de insights sobre percepção dos colaboradores.

A solução permite acelerar a análise de pesquisas de engajamento e apoiar decisões de melhoria organizacional.

### Aconselhador de Plano de Desenvolvimento Individual - PDI

Desenvolvimento de sistema inteligente para sugerir Planos de Desenvolvimento Individual com base em avaliações de desempenho.

Principais características:
