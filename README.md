## Projeto 1: Consultor Digital de RH

Desenvolvimento de assistente inteligente para análise de dados de RH utilizando arquitetura baseada em agentes e Large Language Models.

Principais características:

- Arquitetura multiagente com LangGraph;
- Uso de LLMs para interpretação de perguntas em linguagem natural;
- Integração com banco de dados corporativo via Trino;
- Execução automática de consultas SQL;
- Geração de respostas para apoiar gestores em análises de RH.

O sistema permite que gestores realizem consultas sobre colaboradores, métricas e indicadores utilizando linguagem natural.

## Projeto 2: Sistema de Detecção de Anomalias em Folha de Pagamento

Desenvolvimento de sistema de auditoria automatizada para identificar inconsistências e valores atípicos em dados de folha de pagamento.

Principais características:

- Aplicação de algoritmos de detecção de anomalias, como Isolation Forest;
- Desenvolvimento de pipeline de análise em Python;
- Identificação de padrões incomuns em variáveis financeiras;
- Uso de técnicas de interpretação de modelos, como SHAP;
- Apoio às equipes de RH e auditoria na identificação de possíveis erros ou inconsistências.
