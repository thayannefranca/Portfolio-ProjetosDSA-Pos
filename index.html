Projetos

Simulação de Linhas de Produção com SimPy

Projeto desenvolvido em Python para simular linhas de produção usando SimPy, Pandas e Matplotlib. A simulação utiliza dados reais de produção para gerar métricas de desempenho, backlog, utilização das linhas e gráficos analíticos.

Ver repositório no GitHub

Sistema de Recomendação de Candidatos - ATS

Sistema de recomendação para sugerir candidatos compatíveis com vagas abertas, utilizando embeddings de texto, BERT e similaridade cosseno.

Sistema de Triagem Automatizada de Candidatos - ATS

Solução baseada em LLMs para avaliar a aderência de candidatos aos critérios de uma vaga, com geração de justificativas para apoiar o processo de triagem.

Sistema de Detecção de Anomalias em Folha de Pagamento

Sistema de auditoria automatizada para identificar inconsistências e valores atípicos em dados de folha de pagamento, com uso de algoritmos como Isolation Forest e interpretação com SHAP.

Consultor Digital de RH

Assistente inteligente para análise de dados de RH, utilizando arquitetura multiagente com LangGraph, LLMs e integração com banco de dados corporativo via Trino.

Análise de Sentimento em Pesquisas de Engajamento

Pipeline de análise automática de comentários em pesquisas de clima organizacional, utilizando LLMs para extração de tópicos e classificação de sentimentos.

Aconselhador de Plano de Desenvolvimento Individual - PDI

Sistema inteligente para sugerir Planos de Desenvolvimento Individual com base em avaliações de desempenho, utilizando LLMs, agentes especializados e integração com dados corporativos.
